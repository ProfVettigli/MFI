CLIL / ENS Linus Torvalds

The story of a practical man who gave away his operating system for free, and ended up ruling the Internet.

1. The Visionaries vs The Pragmatist

When we think about the pioneers of the digital revolution, two names instantly come to mind: Steve Jobs (Apple) and Bill Gates (Microsoft). They were unparalleled visionaries and formidable entrepreneurs. They knew how to sell a dream, build closed ecosystems, and create multi-billion dollar business empires based on proprietary software.

Then, there is a Finnish university student named Linus Torvalds. Linus is fundamentally different. He is not a marketer or a charismatic CEO; he is a down-to-earth, highly practical nerd. He didn't want to change the world or become a billionaire. He just wanted a working operating system for his personal computer.

Jobs & Gates

  • Business Mindset: Maximizing profits and market shares.
  • Secrecy: The source code is an industrial secret.
  • Patents: Aggressively patented every software feature.
  • Goal: Building massive corporate monopolies.

Linus Torvalds

  • Engineering Mindset: Maximizing efficiency and performance.
  • Openness: The source code belongs to humanity.
  • Collaboration: Allowed millions of strangers to edit his code freely.
  • Goal: "Just for fun" (as stated in his autobiography).

2. The Birth of Linux (1991)

In 1991, frustrated because he couldn't afford a high-end UNIX system, Torvalds began writing his own operating system kernel as a simple hobby. On August 25, 1991, he posted a now-legendary message on a Usenet newsgroup:

"Hello everybody out there... I'm doing a (free) operating system (just a hobby, won't be big and professional like gnu) for 386(486) AT clones. [...]"

He named it Linux (Linus + UNIX). Instead of selling it, he uploaded the Source Code to the internet, embracing the Open Source philosophy. He invited developers worldwide to find bugs and suggest improvements. What started as "just a hobby" exploded. Driven by a global community of volunteers, Linux evolved at a terrifying speed, becoming far more stable and secure than any commercial alternative. Today, Linux runs all Android phones, most supercomputers, server farms, smart TVs, and the internet infrastructure itself.

3. The Second Revolution: Git and GitHub

Creating the most important Operating System in the world was not enough. In 2005, Torvalds faced a practical problem: too many developers were sending him code changes for Linux, and managing all those files was becoming a nightmare.

Over a single weekend, because he was unhappy with existing tools, he sat down and programmed Git. Git is a Version Control System: a tool that tracks every single modification made to a file, allowing thousands of programmers to work on the exact same project simultaneously without overwriting each other's work.

What about GitHub?

While Linus created Git (the underlying technology), he did not create GitHub. GitHub is a commercial website (a hub), created by third-party developers, designed to host millions of Git repositories in the cloud with a nice graphical interface. It became the "Facebook of Programmers".

The Ultimate Irony: In 2018, Microsoft—the ultimate champion of proprietary software—purchased GitHub for a staggering $7.5 Billion! Microsoft currently owns the largest platform hosting Open Source code, ironically heavily relying on Torvalds' own invention.

4. The Open Source Victory

For over a decade, Torvalds' open-source philosophy was engaged in a holy war against proprietary software. In 2001, Microsoft's CEO Steve Ballmer famously stated: "Linux is a cancer that attaches itself in an intellectual property sense to everything it touches."

🏆

Fast Forward to Today

The war is over. Open Source won aggressively. Today, the world's most profitable, multi-billion dollar companies (including Google, Amazon, Facebook, and yes, even Microsoft) must absolutely rely on Linux. Microsoft's own cloud computing platform, Azure, runs more Linux servers than Windows servers! The practical, no-nonsense approach of a Finnish student proved to be far more scalable, secure, and democratic than any closed, profit-driven corporate lab.

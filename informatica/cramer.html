Risolvere Sistemi: Metodo di Cramer 🧮

Scopri come unire algebra lineare e algoritmi per risolvere sistemi di equazioni in modo automatico.

1. Il Problema: Sistemi di Equazioni

Un sistema di due equazioni lineari in due incognite (x e y) si presenta nella forma:

{ a₁x + b₁y = c₁
{ a₂x + b₂y = c₂

Risolvere questo sistema significa trovare i valori di x e y che soddisfano contemporaneamente entrambe le equazioni. Perché è importante? In ingegneria, economia, fisica e persino nella grafica dei videogiochi, dobbiamo costantemente trovare punti di intersezione o equilibri tra forze diverse. Saper automatizzare questa risoluzione è fondamentale.

2. Il Diagramma di Flusso

L'algoritmo calcola tre determinanti (D, Dx, Dy) e verifica se il sistema è risolvibile.

3. Lo Pseudocodice

PSEUDOCODICE INIZIO LEGGI a1, b1, c1, a2, b2, c2 D ← (a1 * b2) − (a2 * b1) Dx ← (c1 * b2) − (c2 * b1) Dy ← (a1 * c2) − (a2 * c1) SE D ≠ 0 ALLORA x ← Dx / D y ← Dy / D SCRIVI "Soluzione unica: ", x, y ALTRIMENTI SE Dx = 0 E Dy = 0 ALLORA SCRIVI "Sistema Indeterminato" ALTRIMENTI SCRIVI "Sistema Impossibile" FINE SE FINE SE FINE

4. Implementazione in Python

PYTHON def risolvi_cramer(a1, b1, c1, a2, b2, c2): # Calcolo determinanti D = a1 * b2 - a2 * b1 Dx = c1 * b2 - c2 * b1 Dy = a1 * c2 - a2 * c1 if D != 0: x = Dx / D y = Dy / D return f"x = {x}, y = {y}" else: if Dx == 0 and Dy == 0: return "Sistema Indeterminato" else: return "Sistema Impossibile" # Test print(risolvi_cramer(1, 1, 5, 1, -1, 1)) # Esempio: x+y=5, x-y=1

Equazioni di 2° Grado

Dall'algebra al codice: come tradurre la formula del discriminante in un algoritmo decisionale universale.

1. L'Equazione di Secondo Grado

Un'equazione di secondo grado si presenta nella forma normale:

\[ax^2 + bx + c = 0\]
Risolverla significa trovare i valori di \(x\) che rendono vera l'uguaglianza. In informatica, questo è un problema classico di struttura condizionale, poiché il numero di soluzioni dipende dal valore del discriminante (\(\Delta\)).

2. Il Discriminante (\(\Delta\))

Il cuore dell'algoritmo è il calcolo di \(\Delta = b^2 - 4ac\). In base al segno del Delta, l'algoritmo deve prendere tre strade diverse:

\(\Delta > 0\)

Due soluzioni reali e distinte.

\(\Delta = 0\)

Due soluzioni reali e coincidenti.

\(\Delta < 0\)

Nessuna soluzione reale (soluzioni complesse).

3. Diagramma di Flusso e Pseudocodice

📐 Flowchart

📝 Pseudocodice

logic
INIZIO LEGGI a, b, c SE a == 0 ALLORA SCRIVI "Non è di 2° grado" ALTRIMENTI delta ← b*b - 4*a*c SE delta > 0 ALLORA x1 ← (-b + radice(delta)) / (2*a) x2 ← (-b - radice(delta)) / (2*a) SCRIVI x1, x2 ALTRIMENTI SE delta == 0 ALLORA x ← -b / (2*a) SCRIVI "Sol. coincidente: ", x ALTRIMENTI SCRIVI "Nessuna soluzione reale" FINE SE FINE SE FINE

👨‍💻 Sfida: E per le Disequazioni?

Risolvere una disequazione (es. \(ax^2 + bx + c > 0\)) richiede un ulteriore passaggio logico nel tuo algoritmo. Dopo aver trovato le radici, il programma deve decidere se l'intervallo è interno o esterno.

Suggerimento per l'implementazione:
  • Controlla il segno di \(a\)
  • Controlla il verso (\(>\) o \(<\))
  • Verifica \(\Delta > 0, \Delta = 0\) o \(\Delta < 0\)
  • Usa la regola del trinomio!

5. Implementazione in Python

python
import math def solve_quadratic(a, b, c): if a == 0: return "Non è un'equazione di 2° grado" delta = b**2 - 4*a*c if delta > 0: x1 = (-b + math.sqrt(delta)) / (2*a) x2 = (-b - math.sqrt(delta)) / (2*a) return f"Due soluzioni: {x1}, {x2}" elif delta == 0: x = -b / (2*a) return f"Soluzione unica: {x}" else: return "Nessuna soluzione reale"

Che cos'è l' Informatica?

La scienza dell'informazione automatizzata: non solo saper usare il computer, ma saper creare con esso.

1. Definizione e Origini

Il termine Informatica nasce dall'unione di "Informazione" e "Automatica". È lo studio sistematico della memorizzazione, elaborazione e trasmissione dell'informazione.

Contrariamente a quanto si pensa, l'informatica non riguarda solo i computer. Come disse Edsger Dijkstra: "L'informatica non riguarda i computer più di quanto l'astronomia riguardi i telescopi."

2. Il Cuore dell'Informatica: L'Algoritmo

Un algoritmo è una sequenza finita di passi precisi per risolvere un problema. È come una ricetta di cucina, ma scritta in modo che una macchina possa eseguirla senza ambiguità.

// Come pensa un programmatore
function risolvereProblema(problema) {
  if (problema.èDifficile) {
    dividiInProblemiPiùPiccoli();
  } else {
    return soluzioneDiretta;
  }
}

3. Perché studiarla oggi?

Oggi l'informatica è la "nuova alfabetizzazione". Capire come funzionano gli algoritmi che governano i social media, le banche e la medicina è fondamentale per essere cittadini consapevoli. Non si tratta di imparare a programmare per diventare programmatori, ma per imparare a pensare in modo logico e strutturato (Pensiero Computazionale).
